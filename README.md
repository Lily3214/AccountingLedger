# Accounting Ledger Application
This Accounting Ledger application built with Java can track all financial transactions for businesses and individuals.

# Features
<li>
Home Screen
  </li>
  <li>
Add Deposit - prompt user for the deposit information and save it to the csv file
  </li>
    <li>
Make Payment (Debit) - prompt user for the debit information and save it to the csv file
  </li>
  <li>
Ledger - display the ledger screen
  </li>
   <li>
All - Display all entries
  </li>
     <li>
Deposits - Display only the entries that are deposits into the account
  </li>
       <li>
Payments - Display only the negative entries (or payments)
  </li>
       <li>
Reports - A new screen that allows the user to run pre-defined reports or to run a custom search
  </li>
         <li>
Month To Date - Display data by month-to-date.
  </li>
         <li>
Previous Month - Display Last month's data.
  </li>
         <li>
Year To Date - Display data by year-to-date.
  </li>
         <li>
Previous Year - Display Last year's data.
  </li>
           <li>
 Search by Vendor - prompt the user for the vendor name and display all entries for that vendor
  </li>
